Open Core Legacy Patcher

I had an old 2014 Macbook Pro that was only able to upgrade to macOS Big Sur laying around collecting dust. With Open Core Legacy Patcher it has now been upgraded to macOS Sonoma.

Homebridge

This is my second favorite project with a Raspberry Pi. I used a Raspberry Pi 0 for this project. What makes it my favorite is it allows me to integrate my Google Home accessories with Apple Homekit and control it with my Apple Homepod.

Magic Mirror

By far my most favorite home project. I didn't use a mirror for my Magic Mirror but instead an old monitor and built a wooden frame around it. I did have to remove the plastic shell around the monitor screen and then frame it.
